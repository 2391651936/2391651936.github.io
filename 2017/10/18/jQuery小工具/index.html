1、使 jQuery 与其他库并存

首先用 <script></script> 加载 jQuery 之外的库。然后加载 jQuery 库，取得对 $ 的使用权。接着，调用jQuery.noConflict();方法让出 $ ，以便将 $ 的控制权交还给别的库。但是在使用 jQuery 的地方必须用 jQuery 而不是 $ 来调用。

2、jQuery 自定义选择符

我们想要从匹配的带有 horizontal 类的 div 集合中选择第2个项，那么应该使用下面的代码：

$("div .horizontal:eq(1)");

3、filter() 函数

filter() 的功能也十分强大，因为它可以接受函数参数。通过传入的函数，可以执行复杂的测试，以决定相应元素书否应该保留在匹配的集合中。
