2、安装mysql

软件包设置对话框”中输入mysql中”root”用户的密码以及确认密码

3、如何判断mysql是否安装成功

如果mysql启动成功，处于运行状态说明mysql安装成功。

4、登录mysql

输入mysql中”root”用户的密码，登录成功。

5、创建用户:

说明：

-username - 你将创建的用户名, host - 指定该用户在哪个主机上可以登陆,如果是本地用户可用localhost, 如果想让该用户可以从任意远程主机登陆,可以使用通配符%. password - 该用户的登陆密码,密码可以为空,如果为空则该用户可以不需要密码登陆服务器.

例子：

  • CREATE USER ‘dog‘@’localhost’ IDENTIFIED BY ‘123456’;
  • CREATE USER ‘pig‘@’192.168.1.101’ IDENDIFIED BY ‘123456’;
  • CREATE USER ‘pig‘@’%’ IDENTIFIED BY ‘123456’;
  • CREATE USER ‘pig‘@’%’ IDENTIFIED BY ‘’;
  • CREATE USER ‘pig‘@’%’;

6、授权:

说明：

privileges - 用户的操作权限,如SELECT , INSERT , UPDATE 等(详细列表见该文最后面).如果要授予所有的权限则使用ALL.;databasename - 数据库名,tablename-表名,如果要授予该用户对所有数据库和表的相应操作权限则可用表示, 如.*.

例子：

  • GRANT SELECT, INSERT ON test.user TO ‘pig‘@’%’;
  • GRANT ALL ON *.* TO ‘pig‘@’%’;

7、修改监听ip

在目录/etc/mysql下找到my.cnf，用vim编辑，找到my.cnf里面的

bind-address = 127.0.0.1

将其注释。

# bind-address = 127.0.0.1

并添加

bind-address=0.0.0.0

9、mysql 本来是不忽略大小写的，配置 mysql 使 mysql 忽略大小写

打开 vim /etc/mysql/mysql.cnf

10、重启mysql

11、远程登录

用创建的用户名和密码登录

12、说明

mysql默认端口号为3306

mysql的root帐号不允许做远程登录的帐号

13、修改mysql数据库用户密码

格式：mysql> set password for 用户名@localhost = password(‘新密码’);

例子：mysql> set password for root@localhost = password(‘123’);

刷新权限（必须步骤）：flush privileges;　。

14、windows 远程登录

  • use mysql;
  • GRANT ALL PRIVILEGES ON . TO ‘root‘@’%’’ IDENTIFIED BY ‘root’ WITH GRANT OPTION; （第一个’root’代表：用户名，第二个’root’代表：密码）
  • flush privileges; （刷新权限）
  • select host,user from user; （查看是否成功）

15、python3 安装 mysqlclient 出现 OSError: mysql_config not found

执行apt-get install libmysqlclient-dev命令

2、安装PostgreSQL

系统会提示安装所需磁盘空间，输入”y”，安装程序会自动完成。 安装完毕后，系统会创建一个数据库超级用户“postgres”, 密码为空。这个用户既是不可登录的操作系统用户，也是数据库用户。
