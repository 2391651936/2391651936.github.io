最后一项显示的是 pid 和对应的名称

2、杀掉对应的进程

这样使用该端口的进程被杀死了。

  1. 如果当前只有 lo 一个回环网卡的话，设置 eth0 的 ip，使用 ifconfig eth0 192.168.158.2 当前虚拟机网段，根据在虚拟机中设置的 网络连接模式 来区分。临时改变 ip 地址

    1. 桥接模式：使用物理机的网卡进行通讯，但是需要占用物理机网段的 ip 地址，不光可以与自己的主机通讯，还可以与局域网内的其他计算机进行通讯。

    2. NAT 模式：虚拟机使用 VMNAT8 虚拟网卡与主机通讯，如果主机能够上网的话，虚拟机也可以。

    3. HOST-ONLY模式：虚拟机使用 VMNAT1 虚拟网卡与主机通讯，且只能与主机通讯。

  2. -rw-r–r–：分为两部分，

    1. -：文件类型：

      1. -：文件
      2. d：文件夹
      3. l：软连接
    2. rw-r–r–：
      rw-|r–|r–
      :-:|:-:|:-:|
      u|g|o
      u：所有者|g：所属组|o：其他人
      r：读|w：写|x：执行
