centos 安装 mysql

一、安装 mysql

0、这里使用的版本： mysql8.0

1、下载对应的 rpm 文件

下载地址：https://dev.mysql.com/downloads/repo/yum/

下载到本地需要使用 lrzsz 或者 xftp 等别的工具上传到服务器上。

或者在服务器上使用 wget http://dev.mysql.com/get/mysql80-community-release-el7-1.noarch.rpm 进行下载。

2、安装刚刚下载好的 rpm 文件

rpm -ivh mysql80-community-release-el7-1.noarch.rpm

3、安装 mysql

yum -y install mysql-community-server

4、启动 mysql

systemctl start mysqld.service

5、在启动日志中查看默认密码

打开 mysql 日志：vim /var/log/mysqld.log

如上所示：红色位置显示 root 密码 mpq_m<7Vhzor

由于使用的是 mysql8.0 所以在修改 root 密码的时候尤其麻烦。

alter user'root'@'localhost' IDENTIFIED BY 'new password';，mysql8.0以上密码策略限制必须要大小写加数字特殊符号。

我比较烦 root 用户密码还那么麻烦，所以接下来修改密码测试，使 mysql 可以接受 123456 作为密码。

查看当前密码测试：SHOW VARIABLES LIKE 'validate_password%';

  1. set global validate_password.policy = LOW; 设置当前密码安全策略为最低。
  2. set global validate_password.length = 1; 设置当前密码长度要求为1。
  3. alter user'root'@'localhost' IDENTIFIED BY '123456'; 重新修改密码为 123456
