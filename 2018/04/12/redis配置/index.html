redis配置

发表于 2018-04-12 | 分类于 redis

flushdb 删除当前数据库，flushall 删除所有数据库

一、windows 配置

1、下载redis

从 redis 的 github 上下载 .zip https://github.com/ServiceStack/redis-windows

2、在 C 盘创建一个 redis文件夹，然后把之前的 zip 文件解压

3、对 Redis 进行配置

设置 redis 的最大存储容量

设置 redis 的密码

4、把 redis 添加进 windows 的服务

redis-server.exe --service-install redis.windows.conf --loglevel verbose

5、启动 redis 服务

redis-server --service-start

6、卸载 redis 服务

redis-server --service-uninstall

二、linux 配置

1、下载 redis

从 redis 的官网上 https://redis.io/ 上进行下载 .tar.gz 的格式

2、解压

使用 tar -zxvf 命令进行解压

3、安装

进入解压后的文件夹，执行 make 命令进行安装。安装完成后，会在当前文件夹下生成一个 src 的文件夹， src 下包含 redis.conf 配置文件、redis-server 服务器运行命令、redis-cli 连接 redis 客户端。

4、配置 redis 后台运行

建议重新复制一个 redis.conf 文件，再进行修改。

把新复制的文件中，daemonize 修改为 yes 然后执行 redis-server myredis-conf （ps：这里的 myredis-conf 是我重新复制的文件名）。

三、基本语法

1、key 键

2、String 字符串

3、List 列表（单 key 多 value）

4、Set 集合（用来去重）

5、hash 哈希 kv 模式不变，但 value 是一个键值对
