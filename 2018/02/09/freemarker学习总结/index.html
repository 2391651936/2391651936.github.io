freemarker学习总结

发表于 2018-02-09 | 分类于 java

一、freemarker 模板文件(*.ftl)的基本组成部分

  1. 文本：直接输出的内容部分
  2. 注释：不会输出的内容，格式为<#-- 注释内容 -->
  3. 取值(插值)代替输出数据模型的部分，格式为${数据模型}或#{数据模型}
  4. ftl指令：freemarker 指令，类似于 html 标记。(不是很了解在什么地方使用)

二、freemarker 语法及使用方法

  1. 取值(插值)指令及适用类型。

    1. ${var}
      • 适用类型：java 中常用八大基本数据类型以及String应用类型，但是：freemarker 中 boolean 类型显示时，true–>yes，false–>no。
    2. ${var!}
      • 适用类型：对 null 或者不存在的队形进行取值，可以设置默认值，
        例如：${var!”我是默认值”}，
        即：有值时显示正常值，无值时显示默认值。
        可以用来处理表单的回显。
    3. ${封住对象.属性}
      • 适用类型：对封装的对象进行取值，
        例如：${sysUser.name}
    4. ${data?string(‘yyyy-MM-dd’)}
      • 适用类型：对日期格式进行取值
    5. ${var?html}
      • 适用类型：转移 html 内容
    6. <#assign num = 100 />
      • 适用类型：定义变量，支持计算和赋值

    7. 对 List 集合遍历

    8. 类似 python range 的功能

      • 2..5 会生成一个 list [2, 3, 4, 5]，然后可以用他来进行遍历

    9. 对Map集合进行遍历

      • 对 key 进行遍历

      • 对 value 进行遍历

  2. 条件判断指令：

    1. if

    2. if - else

    3. if - elseif - else

    4. switch –常与case break default一起使用 参数可为字符串

  3. 常用内建函数

处理字符串：

函数名 作用
substring 截取字符串，包头不包尾（下标）
cap_first 第一个字母大写
end_with 以什么字母结尾
contains 是否包含目标字符串
date datetime time 转换成日期格式
starts_with 以什么字母开头
index_of 返回某个指定的字符串值在字符串中首次出现的位置（下标）
last_index_of 获取指定字符出现的最后位置（下标）
last_index_of 获取指定字符出现的最后位置（下标）
split 分隔
trim 去两端空格

处理数字：

函数名 作用
x?string(“0.##”) 变成小数点后几位
round 四舍五入
floor 去掉小数点
ceiling 近1变成整数

处理list：

函数名 作用
first 取List值第一个值
last 取List值最后一个值
seq_contains 是否包含指定 object
seq_index_of 指定 object 所在位置
size 集合大小
reverse 集合倒序排列
sort 对集合进行排序
sort_by 根据某一个属性排序
chunk 分块处理
_index 获取当前下标

其他：

函数名 作用
is_string 是否为字符类型
is_number 是否为整数类型
is_method 是否为方法
has_content 判断对象是否为空或不存在
eval 求值
